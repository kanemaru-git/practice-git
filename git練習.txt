gitの使い方
add commit push

commitでコメントし忘れた時(-m 'コメント'）
① ターミナルでgit commitを入力する 
② Vimエディタが立ち上がる 
③ 半角英数字に入力を切り替える 
④「i」を入力する （挿入モードになる） 
⑤ コミットメッセージを入力する 
⑥「esc」を押す （ノーマルモードに戻る） 
⑦「:wq」を入力してエンターを押す (commitメッセージを保存してエディタを閉じる)

変更履歴　git log
kで上、ｊで下、ｑで終了
git log --oneline： 各コミットを1行で簡潔に表示する
git log -n 数字：　数字の数だけコミット表示
他にもたくさんのオプションがある

git pull githubにcommitされたものをローカルにpull

clone ファイル一式をローカルに一気にcloneできる
新しくプロジェクトに参加するときなど
